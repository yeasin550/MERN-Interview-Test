## Getting Started

### Installation

Clone the project:

```bash
  git clone https://github.com/yeasin550/MERN-Interview-Test.git
```

Go to the project directory:

```bash
  cd client
```

Install dependencies:

```bash
  npm install
```

Start the server:

```bash
  npm run dev
```
